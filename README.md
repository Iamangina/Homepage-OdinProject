# Homepage-OdinProject

https://iamangina.github.io/Homepage-OdinProject/

This project contains a custom CSS stylesheet designed for a Pokémon-themed personal portfolio or character page. It focuses on visual presentation, responsive layout, and custom typography inspired by the Pokémon universe.

Features:

- Custom Fonts

- Responsive Design

- Mobile-first layout

- Tablet and desktop optimizations using media queries

- Themed Visual Style

- Fixed background image

- Flexible Layouts

- Flexbox-based structure

- Adaptive sections for content, cards, and footer
